PIANO가 뭔가요?
PIANO는 AI 기반 암호화폐 선물 자동매매 소프트웨어예요. 고객님의 Binance 계정에 API를 연결하면, AI가 24시간 자동으로 매매를 실행해요. 코인 지식이 없어도 사용할 수 있어요.
정말 코인 지식이 없어도 되나요?
네, 괜찮아요! PIANO는 모든 매매 결정을 AI가 대신 해요. 고객님은 Binance 계정만 만들고 API를 연결하면 끝이에요. 차트 분석, 매매 타이밍 포착 등은 모두 AI가 알아서 처리해요.
원금이 얼마나 필요한가요?
최소 50만원부터 시작할 수 있어요. 권장 금액은 500만원 이상이에요. 원금이 클수록 복리 효과가 커지고, 수수료 대비 효율이 좋아져요.
예시:
• 50만원: 월 이용료 대비 부담이 커서 비효율적
• 500만원: 적정 수준, 수익률 3% 시 월 15만원 수익 가능
• 5,000만원: 최적, 수익률 3% 시 월 150만원 수익 가능
PIANO는 투자 상품인가요?
아니요, 투자 상품이 아니에요. PIANO는 소프트웨어 임대(SaaS) 서비스예요. Netflix나 YouTube Premium처럼 매월 이용료를 내고 소프트웨어를 사용하는 구조예요.

고객님의 돈은 고객님의 Binance 계정에 그대로 있고, 우리는 그 계정에서 매매만 대신 실행하는 소프트웨어를 제공해요.
내 돈을 맡기는 건가요?
절대 아니에요! 고객님의 돈은 100% 고객님의 Binance 계정에만 있어요. 우리는 API를 통해 매매 신호만 보낼 뿐, 돈을 직접 보관하거나 관리하지 않아요.

게다가 API 설정 시 출금(Withdraw) 권한은 절대 주지 마세요. 거래(Trade)와 조회(Read) 권한만 있으면 돼요.
어떻게 시작하나요?
정말 간단해요! 3단계만 거치면 돼요:
  1. Binance 계정 생성 (이미 있다면 Skip)
  2. API Key 발급 (Trade + Read 권한만, Withdraw 권한은 제외)
  3. PIANO에 API 연결 (우리 대시보드에서 간단 설정)
이후로는 AI가 24시간 자동으로 일해요!
API Key가 유출되면 위험하지 않나요?
출금 권한만 주지 않으면 안전해요.

API Key를 만들 때 반드시 다음 설정을 확인하세요:
  • Enable Reading (조회 권한: ON)
  • Enable Spot & Margin Trading (거래 권한: ON)
  • Enable Withdrawals (출금 권한: OFF - 절대 켜지 마세요!)
출금 권한이 없으면 API가 유출되어도 돈을 빼갈 수 없어요.
중간에 멈추거나 수동으로 개입할 수 있나요?
네, 언제든지 가능해요!
  • 일시정지: 대시보드에서 "자동매매 중지" 버튼 클릭
  • 수동 개입: Binance 앱/웹에서 직접 포지션 종료 가능
  • API 연결 해제: Binance에서 API Key 삭제하면 즉시 중단
다만 AI 알고리즘을 신뢰하고 장기 운영하는 것을 권장해요.
어떤 코인을 거래하나요?
주로 비트코인(BTC), 이더리움(ETH) 같은 메이저 코인 선물을 거래해요. 유동성이 높고 안정적인 코인 위주로 운영돼요.

알트코인은 변동성이 너무 커서 위험 관리가 어렵기 때문에 제외했어요.
레버리지는 어떻게 설정하나요?
PIANO는 기본적으로 3배 레버리지로 운영돼요. 설정에서 본인이 조절할 수 있어요.
주의: 레버리지가 높을수록 수익도 크지만 손실도 커져요. 처음에는 1~3배 시작을 추천해요.
이용료가 얼마인가요?필독
월 396,000원 (VAT 포함)이에요.

이건 소프트웨어 임대료일 뿐, 투자 원금이 아니에요. 투자 원금은 별도로 고객님의 Binance 계정에 입금하셔야 해요.
실제 필요 금액 예시:
• 소프트웨어 이용료: 396,000원/월
• 투자 원금: 5,000,000원 (권장)
• 총 초기 비용: 약 5,396,000원
무료 체험이 있나요?
네! 7일 무료 체험을 제공해요.

7일 이내에 만족하지 못하면 전액 환불해드려요. 이유를 묻지 않고 바로 환불 처리돼요.
중도 해지하면 환불되나요?
네, 일할 계산으로 환불해드려요.
환불 계산 방식:
환불액 = (월 이용료 ÷ 30일) × 미사용 일수

예시:
월 이용료 396,000원을 결제한 후 10일 사용하고 해지한 경우
환불액 = (396,000 ÷ 30) × 20일 = 264,000원
거래 수수료는 따로 있나요?
PIANO 이용료 외에 Binance 거래 수수료가 별도로 발생해요.
  • Maker 수수료: 약 0.02%
  • Taker 수수료: 약 0.04%
  • 자금조달비용(Funding Rate): 8시간마다 변동
월 50회 거래 기준으로 약 5~10만원 정도 예상돼요.
PIANO는 금융투자업 허가를 받았나요?중요
아니요, 받지 않았어요. 그리고 받을 필요도 없어요.

PIANO는 소프트웨어 제공 서비스이지, 금융투자상품이 아니에요. 우리는 고객의 돈을 운용하거나 투자 조언을 하는 것이 아니라, 단지 자동매매 "도구"를 제공할 뿐이에요.
법적 근거:
자본시장법상 금융투자업은 "타인의 재산을 운용·관리"하는 경우에만 해당돼요. PIANO는 고객이 직접 소유한 API를 통해 고객 본인의 계정에서 거래하므로, 금융투자업에 해당하지 않아요.
투자 손실이 나면 보상받을 수 있나요?필독
아니요, 보상받을 수 없어요.

PIANO는 소프트웨어일 뿐이고, 투자 수익을 보장하지 않아요. 시장 변동에 따른 손실은 고객님의 책임이에요.

다만, 소프트웨어 오류로 인한 손실(예: 버그로 잘못된 주문 발생)이 명백히 입증되면 개별 협의를 통해 해결해드려요.
반드시 기억하세요:
암호화폐 투자는 고위험 투자예요. 원금 전액을 잃을 수도 있어요. 여유 자금으로만 투자하세요.
개인정보는 어떻게 관리되나요?
고객님의 개인정보와 API Key는 암호화되어 한국 내 서버에 안전하게 보관돼요.
  • 수집 정보: 이름, 이메일, 전화번호, Binance API Key
  • 보관 기간: 계약 종료 후 3개월 (법정 기간)
  • 제3자 제공: 고객 동의 없이 절대 제공하지 않아요
자세한 내용은 개인정보처리방침을 참고하세요.
계약서는 어떻게 되나요?
회원가입 시 소프트웨어 이용약관에 동의하시면 계약이 체결돼요.

이용약관에는 다음 내용이 포함돼요:
  • 서비스 제공 범위
  • 이용료 및 환불 정책
  • 책임의 한계
  • 계약 해지 조건
전문은 이용약관에서 확인하세요.

실제 손실 시나리오:
원금 500만원, 5배 레버리지 사용 시
• 시장 20% 급락 → 원금 100% 손실 (청산)
• 시장 10% 하락 → 원금 50% 손실
• 시장 5% 하락 → 원금 25% 손실
절대 생활비나 급하게 쓸 돈을 투자하지 마세요!
손실을 줄이는 방법이 있나요?
네, 몇 가지 방법이 있어요:
  • 레버리지 낮추기: 1~3배로 설정하면 위험 감소
  • 분할 투자: 한 번에 다 넣지 말고 나눠서 투자
  • Stop-Loss 설정: 자동 손절매 기능 활용
  • 장기 운영: 단기 변동에 흔들리지 말고 최소 6개월 이상 운영
다만 이 방법들도 손실을 완전히 막을 수는 없어요.

  • 시장 환경 변화 (변동성, 트렌드 패턴)
  • 거래소 수수료 정책 변경
  • 유동성 변화
  • 규제 환경 변화

폭락장에서는 어떻게 대응하나요?
PIANO는 Stop-Loss(손절매) 시스템이 내장돼 있어요.

일정 손실률을 넘으면 자동으로 포지션을 종료해서 더 큰 손실을 막아요. 다만 급격한 폭락 시에는 Stop-Loss가 제대로 작동하지 않을 수도 있어요.
