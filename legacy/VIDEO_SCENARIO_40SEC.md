## Post-Production Guide

### Transitions
- Use smooth cross-dissolves (0.3 sec) between scenes
- **Exception**: Scene 1→2 use seamless morph transition on laptop screen

### Typography
- **Font**: Bold modern sans-serif (Inter or SF Pro Display)
- **Colors**:
  - Emphasis: Cyan (#00E5FF)
  - Body: White (#FFFFFF)
- **Size**: Minimum 5% of frame height for readability

### Music Track
40-second loop of modern corporate electronic music:
- **0:00-0:05**: Ambient, subtle (Scene 1)
- **0:05-0:25**: Building optimistically (Scenes 2-6)
- **0:25-0:35**: Upbeat, confident (Scenes 7)
- **0:35-0:40**: Resolution with confidence (Scene 8)

### Sound Design
Layer the following under music:
- UI sounds (soft beeps, clicks)
- Data processing ambiance
- Market ticker sounds
- Success notifications

**Important**: Keep audio professional, not overly "gamified"

### Export Settings
```
Resolution: 1920x1080 (Full HD)
Frame Rate: 30fps
Codec: H.264
Bitrate: High (8-12 Mbps for web)
Format: MP4
Audio: AAC, 192 kbps, Stereo
```

### Accessibility
- Korean subtitles: Minimum 5% of frame height
- High contrast text (white on dark)
- Positioned in bottom third
- Sans-serif font for clarity

## Production Workflow

### Step 1: Generate Individual Clips (Veo 2)
1. Copy each scene's Veo 2 prompt
2. Generate 8 separate video clips
3. Download all clips in highest quality

### Step 2: Consistency Check
- Verify color palette consistency (#0A0E27 background, #00E5FF accents)
- Check for visual continuity between scenes
- Ensure same businessman appearance in Scenes 1 & 8

### Step 3: Editing (Premiere Pro / DaVinci Resolve)
1. Import all 8 clips to timeline
2. Trim each clip to exactly 5 seconds
3. Add transitions between scenes
4. Apply color grading if needed for consistency

### Step 4: Typography (After Effects)
1. Create Korean text overlays for each scene
2. Animate text with subtle fade-in/fade-out
3. Export as transparent overlay

### Step 5: Audio Mix
1. Add music track (40 seconds, loop if needed)
2. Layer sound effects synchronized to visuals
3. Mix levels: Music 60%, SFX 30%, Dialogue 10%

### Step 6: Final Export
1. Render with export settings above
2. Test playback on target devices
3. Optimize file size for web (target <20MB)
