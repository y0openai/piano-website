개요

와이알파 주식회사(이하 "회사")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」, 「정보통신망 이용촉진 및 정보보호 등에 관한 법률」 등 관련 법령을 준수하고 있습니다.

제1조 (개인정보의 수집 항목 및 방법)

1. 수집하는 개인정보 항목

수집 시점 수집 항목 수집 목적
회원가입 및 구독 신청 이름, 이메일 주소, 휴대전화번호, 결제정보(카드번호, 계좌번호) 본인 확인, 서비스 제공, 결제 처리, 고객 문의 응대
API 연결 거래소 API 키, API Secret 자동매매 서비스 제공
서비스 이용 과정 거래 내역, 수익률 데이터, 접속 로그, IP 주소, 쿠키 서비스 제공, 성능 분석, 부정 이용 방지
문의/상담 이름, 이메일, 문의 내용 고객 지원, 민원 처리

2. 개인정보 수집 방법

  • 웹사이트 회원가입 및 구독 신청 시
  • 이메일, 전화를 통한 상담 과정에서
  • 서비스 이용 과정에서 자동 수집 (쿠키, 접속 로그 등)

제2조 (개인정보의 수집 및 이용 목적)

회사는 수집한 개인정보를 다음의 목적으로만 이용합니다:

  1. 서비스 제공: PIANO AI 소프트웨어 임대, 자동매매 기능 제공
  2. 회원 관리: 본인 확인, 개인 식별, 부정 이용 방지
  3. 결제 처리: 구독료 결제, 환불 처리
  4. 고객 지원: 문의 응대, 공지사항 전달, 민원 처리
  5. 서비스 개선: 이용 통계 분석, 서비스 품질 향상
  6. 마케팅: 이벤트 정보 제공 (사전 동의를 받은 경우에 한함)

제3조 (개인정보의 보유 및 이용 기간)

개인정보 항목 보유 기간 근거
회원 기본 정보 구독 해지 후 3개월 서비스 제공 및 분쟁 해결
거래 내역, 수익률 데이터 구독 해지 후 3개월 서비스 제공 및 분석
결제 정보 5년 전자상거래법 (계약 또는 청약철회 등에 관한 기록)
접속 로그 3개월 통신비밀보호법
불만 또는 분쟁 처리 기록 3년 전자상거래법 (소비자 불만 또는 분쟁처리에 관한 기록)

보유 기간 종료 시, 해당 개인정보는 지체 없이 파기됩니다.

제4조 (개인정보의 파기 절차 및 방법)

1. 파기 절차

  • 보유 기간이 경과한 개인정보는 종료일로부터 5일 이내에 파기합니다.
  • 법령에 따라 보존해야 하는 정보는 별도 저장 후 관리합니다.

2. 파기 방법

  • 전자적 파일: 복구 불가능한 방법으로 영구 삭제
  • 종이 문서: 분쇄기로 분쇄하거나 소각

제5조 (개인정보의 제3자 제공)

원칙적으로 제3자 제공 없음

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

  1. 이용자가 사전에 동의한 경우
  2. 법령에 의거하거나 수사 목적으로 법령에 정해진 절차와 방법에 따라 요구가 있는 경우

제3자 제공 현황 (결제 처리 목적)

제공받는 자 제공 목적 제공 항목 보유 기간
PG사 (결제 대행사) 결제 처리 이름, 결제정보 거래 종료 후 5년

회사는 서비스 제공을 위해 다음과 같이 개인정보 처리를 위탁하고 있습니다:

수탁업체 위탁 업무 보유 기간
Amazon Web Services (AWS) 서버 호스팅, 데이터 저장 위탁 계약 종료 시까지
PG사 결제 대행 위탁 계약 종료 시까지

제8조 (개인정보의 안전성 확보 조치)

회사는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다:

  1. 관리적 조치: 내부관리계획 수립·시행, 정기적 직원 교육
  2. 기술적 조치:
    • 개인정보의 암호화 저장 및 전송
    • 해킹 방지를 위한 보안 프로그램 설치 및 주기적 점검
    • 접근 권한 관리
    • 접속 기록 보관 및 위·변조 방지
  3. 물리적 조치: 전산실, 자료보관실 등의 접근 통제

제9조 (API 키 보안)

API 키 보안 정책

  1. 이용자의 거래소 API 키는 암호화되어 저장됩니다.
  2. 회사는 API 키에 출금 권한을 요구하지 않습니다.
  3. API 키는 거래 실행 및 조회 목적으로만 사용됩니다.
  4. API 키 유출 시 즉시 회사에 통보하고 거래소에서 키를 삭제하시기 바랍니다.
  5. API 키 유출로 인한 손실은 이용자 본인의 책임입니다.

제10조 (쿠키의 운용)

회사는 이용자에게 개인화된 서비스를 제공하기 위해 쿠키를 사용합니다.

1. 쿠키의 사용 목적

  • 이용자의 로그인 상태 유지
  • 이용자 선호 설정 저장 (언어 설정 등)
  • 웹사이트 방문 및 이용 형태 파악

2. 쿠키의 거부

이용자는 쿠키 설정을 거부할 수 있으나, 일부 서비스 이용에 제한이 있을 수 있습니다.
